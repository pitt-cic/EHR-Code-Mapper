How It Works

Two-stage intelligent pipeline

Combining semantic search with AI reasoning for accurate, explainable mappings

01

Vector Similarity Search

Your proprietary codes are embedded using Amazon Titan Text Embeddings V2 (1024 dimensions) and matched against pre-computed standard code embeddings in AWS S3 Vectors.

  • Automatic acronym expansion via Claude
  • Cosine similarity search returns top 30 candidates
  • Re-enhancement for difficult matches (>0.65 distance)
02

AI-Powered Reasoning

Claude Sonnet 4.5 analyzes candidates using clinical context, test purpose, and data characteristics to select the top 3 matches with detailed explanations.

  • Considers test type and clinical purpose
  • Analyzes average values and categories
  • Uses frequency rank data to break ties
